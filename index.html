Why a blog?

An attempt to explain how does this website exists

It's easy to decide to start a new project. Let's say you want toto learn how to make a mobile application using React Native and publish your first app. First you go through the docs, find a nice tutorial and start reading it. The begining is often very satisfying and the learning curve is quick. But at some point things start to get harder and you find yourself stuck. Then what to do next? Stop that project and start a new thing, of course!

I have been doing that for too long and realized that it's time to take a project until the last step! Therefore, I decided to start this blog as a place that I can write things that I have learned, things that I want to learn and also a place to show some things that I managed to do.
